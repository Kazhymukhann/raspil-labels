# Автозаливка бирок в Google Drive — настройка (один раз, ~5 минут)

После настройки на сайте появится кнопка **«☁ Залить в Drive»**: загрузил XML → нажал → бирки
сами лягут в нужную папку на Google Drive (станок их оттуда забирает).

## Шаги

1. **Узнай ID папки `labels`** на Google Drive.
   Открой папку `labels` в Google Drive, скопируй ссылку. ID — это кусок между
   `/folders/` и концом:
   `https://drive.google.com/drive/folders/`**`1AbCdEf123...`** → ID = `1AbCdEf123...`

2. **Создай скрипт.** Зайди на <https://script.google.com> → **Новый проект**.
   - Удали то, что там есть, и вставь весь код из файла [`Code.gs`](Code.gs).
   - В строке `const PARENT_FOLDER_ID = '...'` вставь свой ID папки из шага 1.
   - Сохрани (значок дискеты).

3. **Разверни как веб-приложение.**
   - Справа вверху **Развернуть → Новое развёртывание**.
   - Тип (шестерёнка) → **Веб-приложение**.
   - «Запуск от имени» → **Я** (твой аккаунт).
   - «У кого есть доступ» → **Все** (Anyone).
   - **Развернуть** → разреши доступ (выбери свой аккаунт, «Дополнительно» → «Перейти … (небезопасно)» → Разрешить — это нормально для своего скрипта).
   - Скопируй **URL веб-приложения** (вида `https://script.google.com/macros/s/AKfy.../exec`).

4. **Пришли мне этот URL.** Я впишу его в сайт (`config.js → driveUploadUrl`), и кнопка заработает.

## Как пользоваться (потом)
На сайте: загрузил XML раскроя → нажал **«☁ Залить в Drive»** → бирки появились в
`labels/<имя из XML>/` на Drive. Старые бирки в этой подпапке заменяются новыми.

## Примечание по безопасности
URL веб-приложения даёт право записи в папку `labels`. Он неочевидный, но всё же —
не публикуй его широко. При желании добавлю проверку по секретному слову.
